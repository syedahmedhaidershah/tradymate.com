# tradymate.com — marketing site

Public static site for **Tradymate**: mobile-first quotes, customers, dashboard, and invoicing for tradespeople and small field teams.

## Live site

**[https://tradymate.com](https://tradymate.com)** (GitHub Pages — this repo’s `main` branch)

## Apps

- **iOS:** [Tradymate on the App Store](https://apps.apple.com/th/app/tradymate/id6749824428)
- **Android:** [Tradymate on Google Play](https://play.google.com/store/apps/details?id=com.emaxsole.tradymatemob&pcampaignid=web_share)

## About this repository

- Contains the **built** HTML, assets, and images served at the apex domain.
- **`CNAME`** is set to `tradymate.com` for GitHub Pages custom domain DNS.
- The marketing app is developed separately; this repo is updated when a new production build is published.
